# Examples

| # | Example | Description |
|---|---------|-------------|
| 01 | [Quickstart](01_quickstart.py) | Assess sovereignty level, get template, run compliance check |
| 02 | [Sovereignty Assessment](02_sovereignty_assessment.py) | Data sensitivity detection and regulatory level mapping |

## Running the examples

```bash
pip install agent-sovereign
python examples/01_quickstart.py
```
